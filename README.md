# Broclean Cleaning Services 🧹

אתר מקצועי לחברת ניקיון **Broclean Cleaning Services** — בנוי עם Next.js 15, React 19, TypeScript, Tailwind CSS ו-Framer Motion.

## 🚀 טכנולוגיות

- **Next.js 15** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** — אנימציות עדינות
- **Lucide Icons**
- **פונט Rubik** — עברית מלאה

## 📁 מבנה הפרויקט

```
src/
├── app/
│   ├── layout.tsx      # Layout ראשי + Metadata + Schema.org
│   ├── page.tsx        # עמוד הבית
│   ├── globals.css     # סגנונות גלובליים + RTL
│   ├── sitemap.ts      # Sitemap אוטומטי
│   └── robots.ts       # Robots.txt
└── components/
    ├── Navbar.tsx       # ניווט עם Sticky + Mobile Menu
    ├── Hero.tsx         # Hero Section מרשים
    ├── Services.tsx     # 6 שירותים בכרטיסים
    ├── WhyUs.tsx        # למה לבחור בנו
    ├── Gallery.tsx      # גלריה לפני/אחרי
    ├── Testimonials.tsx # המלצות לקוחות
    ├── CTABanner.tsx    # באנר קריאה לפעולה
    ├── FAQ.tsx          # שאלות ותשובות
    ├── Contact.tsx      # טופס יצירת קשר
    ├── Footer.tsx       # Footer מלא
    └── FloatingButtons.tsx # כפתור WhatsApp צף
```

## ⚡ התקנה והרצה

```bash
npm install
npm run dev
```

פתחו [http://localhost:3000](http://localhost:3000)

## 🌐 Deploy ל-Vercel

```bash
npm run build
```

או חברו את ה-Repository ל-Vercel לדפלוי אוטומטי.

**דומיין:** `broclean-clean.com`
